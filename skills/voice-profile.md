# Skill: Voice Profile

## Purpose

Preserve the user's voice while making the argument clearer.

## Default voice

Use this default when the user has not supplied a custom voice sample:

- Direct
- Calm
- Specific
- Evidence-led
- Not apologetic
- Not inflated
- Short sentences
- Plain language
- No corporate filler
- No motivational slogans

## What to avoid

Do not write:
- "I deserve this"
- "I am passionate about"
- "I would be honored"
- "I just wanted to"
- "I hope this is okay"
- "I have gone above and beyond" unless the evidence shows exactly how
- Corporate filler such as "leverage," "synergy," "circle back," "at the end of the day"

## How to adapt to the user

If the user provides writing samples, infer:
- Sentence length
- Formality level
- How directly they make requests
- Whether they use bullets, paragraphs, or scripts
- Words they naturally use

Then match that style without copying private details into unrelated outputs.

## Output test

Before finalizing any ask or script, ask:

"Would a real person say this out loud in a meeting?"

If no, rewrite it.

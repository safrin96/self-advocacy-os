# Project Skill Cards

These are portable skill cards for Self-Advocacy OS.

They are not hidden Claude or Codex skills. They are plain project files that Claude Code can read because `CLAUDE.md` tells it to use them.

Use them in this order:

1. `research.md` - What do we know? What is missing?
2. `skeptic.md` - What would a decision-maker push back on?
3. `voice-profile.md` - How should this sound?
4. `humanizer.md` - How do we make it sayable in the room?

The point is not to make the user sound more polished. The point is to make the case more evidenced, more honest, and easier to say out loud.
